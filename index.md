## Week 0 — Community Bonding (May 17, 2026)

I'm Kaushik Raja, a sophomore at UIUC studying CS and Crop Sciences.
This summer I'll be parallelizing r.proj and other raster processing
modules in GRASS GIS using OpenMP.

r.proj is one of GRASS's most used modules but runs entirely on one
core. The goal is to change that using a two-path memory architecture
with OpenMP, making raster reprojection significantly faster for
researchers working with large datasets.

Full project details on the GRASS Wiki:
